# Debian DevTools Setup

- 데비안 계열의 OS(ex. Ubuntu)에서 필요한 모든 개발 도구를 다운로드합니다.
- 쉘 스크립트로 작동합니다.
- PR 환영합니다.

## How To Use
- 이 레포지토리를 클론합니다.
- 클론한 디렉토리에서 `<쉘 명> <스크립트 이름>.sh`를 입력합니다. 모든 개발 도구를 다운로드하려면 `all-in-one.sh`를 입력하면 됩니다.
